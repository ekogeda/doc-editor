# doc-editor
A demo on document editing module (PSPDF)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Launch demo
See [Demo](https://doc-editor.netlify.app/).
